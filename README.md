# Inventory and Supply Chain Management Analysis

Interactive Power BI dashboard for analyzing inventory and supply chain operations using KPI tracking, forecasting, and data visualization. The project focuses on monitoring inventory trends, operational efficiency, and order performance to support data-driven decision-making.

## 🚀 Features
- Inventory and supply chain KPI tracking  
- Order and demand trend analysis  
- Interactive dashboards and visualizations  
- Forecasting and performance monitoring  
- Dynamic filtering and reporting  

## 🛠️ Tools Used
- Power BI  
- Power Query  
- DAX  
- CSV Dataset  

## ⚙️ Project Workflow
- Imported and cleaned the dataset using Power Query  
- Created data models and KPIs using DAX  
- Built interactive dashboards and reports  
- Added charts, slicers, gauges, and forecasting visuals  
- Analyzed inventory and supply chain performance metrics  

## 📈 Insights Generated
- Inventory movement and stock trends  
- Order fulfillment performance  
- Supply chain efficiency metrics  
- Forecasting and operational analysis
